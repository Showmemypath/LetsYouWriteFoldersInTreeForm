## Overview 
Helps you brainstorm a folder structure , adds trees and branches . You just need to write the name of the file/folder.
It's useful for when you are brainstorming a folder structure and can't decide on one , edit it manually

## Features
1. Visualize folders in tree form without having the need to write the branches
2. Lightweight and begginer friendly 

## Installation
Clone the repository :
Only need to clone src to get the essentials
```bash
git clone https://github.com/Showmemypath/LetsYouWriteFoldersInTreeForm.git
```

## How to use
- Any text without a . ( full stop / period ) is considered a folder name and any text with a . is a file name 
- Folder names automatically get a / added 
- Write the folder/file name in the textbox
- Enter to display the text in the display area
- RightArrow and LeftArraow to go deeper/outer in the tree 
- Shift + backspace to delete the last entry ( works only when cursor is active on text box )
